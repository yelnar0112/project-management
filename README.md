# Users

### Get all users

1  Endpoint: /users
- Method: GET
- Description: Retrieves a list of all users.
- Create a new user

2 Endpoint: /users
- Method: POST
- Description: Creates a new user.
- Get a specific user by ID

3 Endpoint: /users/{id}
- Method: GET
- Description: Retrieves data for a specific user by their ID.
